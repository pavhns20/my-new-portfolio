Retro Records

Website built with HTML/ CSS/ JavaScript/ PHP/ MySql.

The brief

The brief for this project was to build a website using WordPress for a cake shop called Sweet Treats. The logo was designed in Adobe Illustrator and there were already some branding colours in place. The website needed to have different pages that included a shop page implemented with WooCommerce, with options of variations of products including size & cost. There also needed to be a sign up/ login account page so that returning customers could input their details to access their account and hisory. The site had to have an about page detailing the store owner and had to have a contact page. This site also needed to have a cart and checkout functionality linked to paypal for payments.There also had to be a search function.
