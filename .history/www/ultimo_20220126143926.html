Ultimo Real Estate

Real Estate website built with HTML/ CSS/ JS.

About the project

The brief for this project was to build a website using HTML/ CSS & JavaScript for a Real Estate company based in Ultimo, NSW.
I designed the logo in Adobe Illustrator using a traditional brick red for the "roof".
The website needed to have different pages that included the property of the week, a buy/ sell/ rent page- highlighting available properties, an agents page with contact details for each agent, a suburb profile to inform of the local area, FAQ page with an accordian and a contact page with different means of contact.
This website didn't need to be made responsive.
