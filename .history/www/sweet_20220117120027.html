Sweet Treats

Cake store website built with WordPress.

About the project

The brief for this project was to build a website using WordPress for an online cake store.
I designed the logo in Adobe Illustrator, using the brand colours already being used.
The website needed to have shop functionality implemented with WooCommerce, with product options & variations.
There also needed to be a sign up/ login account facility for returning customers. It also needed to have a cart and checkout functionality linked to paypal for payments.
