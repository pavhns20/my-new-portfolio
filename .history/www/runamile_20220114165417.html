Run A Mile

Online running shoe store built with WordPress.

About the project

The brief for this project was to build a website using WordPress for a running-shoe shop.
I designed the logo in Adobe Illustrator.
The website wasn't going to sell shoes online so didn't need any shop functionality.
The main aim of the website was to give customers the ability to browse online for shoes that they liked, and then go to the store to experience the personalised shoe-fitting service being offered.
The client requested an ocean blues and greens colour palette.
