Australian Design Architects

Architects website built with HTML/ CSS/ JS.

About the project

The brief for this project was to build a website using HTML/ CSS & JavaScript for an Architects company that has a focus on being environmentally conscious, based in Sydney.
I designed the logo in Adobe Illustrator with a nod to the "green" focus of the company.
The website needed to have different pages for the different project types carried out by the company, along with images showcasing their works.
The website also had to have an awards page highlighting their achievements, a contact page with different means of contact and an about the team page.
This website didn't need to be made responsive.
